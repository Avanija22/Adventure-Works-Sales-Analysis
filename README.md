# Adventure Works Sales Analysis (Power BI)

## Project Overview
Developed an interactive Power BI dashboard using the Adventure Works dataset to analyze sales performance, customer behavior, product trends, and regional business metrics.

## Key Features

### Executive Dashboard
- High-level overview of sales performance
- KPI tracking and business performance monitoring
- Interactive filtering and drill-down capabilities

### Geographic Sales Analysis
- Map-based visualization of regional sales performance
- Location-wise business insights
- Identification of top-performing markets

### Product Analysis
- Product-level sales performance tracking
- Revenue and profitability analysis
- Product category comparison

### Customer Analysis
- Customer segmentation and purchasing behavior analysis
- Customer contribution to overall sales
- Identification of high-value customers

### Advanced Analytics
- Decomposition Tree for root-cause analysis
- Interactive Q&A functionality
- Dynamic filtering and cross-report interactions
